## Installation

Clone, install and run my blog. Local development at http://localhost:300

Clone the project

```bash
  git clone https://github.com/lucasangelino/Blog
```

Go to the project directory

```bash
  cd blog
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
